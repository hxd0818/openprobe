## 1. 人物维度定位

### 为什么人物维度不可缺失

KK集团调查的反例证明：缺少人物调研会导致：
- **无法判断企业战略意图** — CEO/创始人的背景决定公司走向
- **无法识别隐性关联** — 高管兼任、投资人网络是资本关系的重要入口
- **竞对对比流于表面** — 只比产品不比团队，错失战略层面的洞察
- **投资逻辑黑箱** — 不知道投资人为什么投这家，就无法判断资本态度

### 人物在螺旋迭代中的位置

```
Phase 0: 种子初始化 → 提取法人/创始人/CEO（基础人物种子）
    ↓
Phase N 每轮迭代:
  ① 回顾评估时同步检查人物维度置信度
  ② 决策时可选择"人物深挖"作为本轮重点方向
  ③ 以人物名为核心设计扩散query
  ④ 提取新人物/新关联实体
  ⑤ 收敛判断时纳入质量门禁检查（关键人物数 ≥ 5）
```

**人物维度不作为独立轮次的唯一焦点，而是融入每轮迭代中。** 当竞对/产业链/资本任一维度涉及具体人名时，自动触发人物深挖。

## 2. 目标企业管理层完整挖掘路径

### 2.1 必挖角色清单（按优先级排序）

| 优先级 | 角色 | 调研必要性 | 理由 |
|--------|------|-----------|------|
| P0 | CEO / 总经理 / 法定代表人 | 必须 | 企业决策核心，战略方向决定者 |
| P0 | 实际控制人 / 创始人 | 必须 | 最终受益人，控制权归属 |
| P0 | 联合创始人（如有） | 必须 | 创业团队完整性，能力互补分析 |
| P1 | 董事长 / 董事（执行董事优先） | 重要 | 治理结构，战略决策参与度 |
| P1 | CFO / 财务负责人 | 重要 | 融资/IPO关键人物，财务健康度的窗口 |
| P1 | 各品牌GM / 业务线负责人 | 重要 | 多品牌企业的实际操盘手 |
| P2 | CTO / 技术负责人 | 推荐 | 技术驱动型企业必须 |
| P2 | CMO / 市场负责人 | 推荐 | 品牌驱动型企业必须 |
| P2 | 监事 | 可选 | 监督角色，有时揭示关联方信息 |

### 2.2 挖掘方法

#### 方法A：百科+泛化搜索（初始名单获取）

```bash
# 百科查询通常包含法人、创始人、高管信息
python3 scripts/investigate-v7.py "公司名" --baike

# 泛化搜索补充
'"{公司名}" 创始人 团队 CEO 总经理 法定代表人'
'"{公司名}" 核心高管 管理层 董事 监事'
'"{公司名}" 联合创始人 创业团队'
```

#### 方法B：工商数据挖掘（最全名单）

```bash
# 工商信息中的高管列表
'"{公司名}" 高管名单 董事会成员 监事会 主要人员 工商'

# 历史变更（谁进谁出 = 重要信号）
'"{公司名}" 高管变更 人事变动 辞职 新任 董事 监事 2024 2025'
```

#### 方法C：新闻/访谈/活动（深度信息）

```bash
# 公开演讲和访谈
'"{公司名}" {人名} 访谈 演讲 采访 观点 发言'
'"{公司名}" {人名} 行业峰会 论坛 嘉宾 分享'

# 社交媒体（LinkedIn/微博/公众号）
'"{人名}" 微博 公众号 LinkedIn 动态'
```

#### 方法D：招股书/年报（上市公司/拟上市企业金矿）

```bash
# 招股书"董监高"章节是最完整的人物信息源
'"{公司名}" 招股书 董监高 核心团队 管理层 简介 履历'
'"{公司名}" 年报 公司治理 董事 监事 高管 简历'
```

### 2.3 最小产出要求

**质量门禁硬性指标：关键人物数 ≥ 5人**

5人的构成建议：
- 1-2人：目标企业核心决策者（CEO + 创始人/实控人）
- 1-2人：目标企业其他关键高管（CFO/CTO/品牌GM等）
- 1-2人：关键投资人代表（领投方合伙人或重要个人投资者）

## 3. 人物调研标准项

对每个确认的关键人物，完成以下调研项：

### 3.1 基础身份

| 字段 | 说明 | 来源 |
|------|------|------|
| 姓名 | 全名 + 常用名/英文名 | 百科/工商/新闻 |
| 当前职务 | 在目标企业中的职位 | 百科/工商 |
| 在企业中的角色标签 | 创始人/职业经理人/投资人代表/家族成员 | 综合判断 |
| 年龄/代际 | 出生年份或年龄段（影响战略风格） | 新闻/访谈推断 |

### 3.2 教育背景

| 字段 | 说明 |
|------|------|
| 最高学历及院校 | 本/硕/博 + 学校名称（名校=校友网络线索） |
| 专业 | 专业方向（技术背景 vs 商科背景 vs 复合） |
| 海外经历 | 是否有海外留学/工作经历（国际化视野信号） |
| 校友关系 | 与同行业其他企业CEO是否为校友（→ 关系网络） |

### 3.3 过往任职履历

| 字段 | 说明 |
|------|------|
| 加入当前企业前的完整职业路径 | 从第一份工作到现在的关键节点 |
| 每段经历的时长和职务 | 判断稳定性 vs 跳槽频率 |
| 前雇主与当前企业的关系 | 是否来自竞对/供应商/客户/投资方 |
| 创业经历（如有） | 之前创过什么业？成功还是失败？ |

**关键分析点**：
- "空降兵" vs "内部培养" → 影响企业文化和决策风格
- 来自竞对的高管 → 可能带来竞对的策略认知
- 连续创业者 → 韧性和资源调动能力更强

### 3.4 行业人脉

| 字段 | 说明 |
|------|------|
| 行业协会任职 | 在行业协会中的角色（会长/理事/会员） |
| 常出席的活动/论坛 | 反映行业影响力圈层 |
| 已知的行业关系 | 与供应商/客户/竞对方高层是否有私交 |
| 媒体曝光度 | 是否经常接受采访、发表公开观点 |

### 3.5 公开言论与风格特征

| 字段 | 说明 |
|------|------|
| 近期公开言论摘要 | 最近12个月的重要表态（战略方向、行业判断） |
| 管理风格标签 | 激进/稳健/务实/愿景型/数据驱动型 |
| 对竞争的态度 | 公开提及竞对时的措辞和姿态 |
| 个人IP强度 | 是否有个人品牌（公众号/书籍/社交媒体粉丝量） |

### 3.6 兼任情况

| 字段 | 说明 |
|------|------|
| 外部董事/监事职务 | 在其他公司担任的职务清单 |
| 投资人角色 | 是否以个人名义投资了其他企业 |
| 协会/社会组织职务 | 在非营利组织中的角色 |
| 兼任的战略含义 | 这些兼任意味着什么？（资源导入/战略协同/精力分散风险） |

### 3.7 股权激励

| 字段 | 说明 |
|------|------|
| 持股比例 | 直接/间接持股数量和比例 |
| 股权来源 | 初始授予/期权行权/增资认购 |
| 股权变化趋势 | 近期增持/减持动向 |
| 与控制权的关系 | 持股是否足以影响公司决策 |

## 4. 竞对企业关键决策者画像

### 4.1 为什么需要竞对CEO画像

- **战略风格对比** — 创始人背景差异往往解释了企业战略差异
- **预判竞对动向** — 了解CEO的个人偏好可以预判其决策倾向
- **寻找差异化机会** — 对手的短板可能是我们的机会

### 4.2 调研范围

对每个 L1 直接竞对，至少完成以下人物的画像：

| 角色 | 必要性 | 说明 |
|------|--------|------|
| CEO / 创始人 / 实控人 | 必须 | 战略决策核心 |
| CFO（如可获取） | 推荐 | 融资能力和财务策略窗口 |

L2/L3 竞对仅需记录 CEO/创始人姓名 + 一句话背景。

### 4.3 竞对高管对比表（报告必含）

| 维度 | 目标企业CEO | L1竞对A CEO | L1竞对B CEO | L1竞对C CEO |
|------|------------|-------------|-------------|-------------|
| 姓名 | | | | |
| 教育背景 | | | | |
| 核心从业经历 | | | | |
| 管理风格 | | | | |
| 行业经验年限 | | | | |
| 公开言论关键词 | | | | |
| 战略风格差异 | （基准） | | | |

**分析要求**：对比表后必须附带解读段落，回答：
1. 目标企业CEO与各竞对CEO的背景差异如何影响了各自公司的战略选择？
2. 哪位CEO的组合最令人警惕？为什么？

## 5. 关键投资人背景与投资逻辑

### 5.1 必挖投资人范围

| 类型 | 范围 | 调研深度 |
|------|------|---------|
| 领投方 | 每轮融资的领投机构/个人 | 深度（见下方标准项） |
| 跟投方 | 单笔投资占比 > 10% 的机构 | 中度（基本信息+投资逻辑） |
| 战略投资人 | 产业资本（同行或上下游企业投资） | 深度（协同效应分析） |
| 个人重要股东 | 持股 > 5% 或担任董事的自然人 | 中度 |

### 5.2 投资人调研标准项

#### 机构投资人

| 字段 | 说明 |
|------|------|
| 机构全称 + 简称 | |
| 机构类型 | VC/PE/产业资本/国资/母基金/天使 |
| 管理规模（AUM） | 判断其在行业中的话语权 |
| 在本项目的负责合伙人 | 具体对接人（→ 人物维度） |
| 投资金额 + 轮次 + 时间 | |
| 投后参与度 | 仅财务回报？还是有董事会席位？ actively involved？ |
| **投资逻辑（核心字段）** | 为什么投这家？看重什么？公开表述的投资逻辑是什么？ |
| **同类投资组合** | 该机构还投了哪些同赛道/上下游企业？（→ 竞对+产业链线索） |
| 退出预期 | IPO/并购/转让？时间表？ |

#### 个人投资人

| 字段 | 说明 |
|------|------|
| 姓名 + 身份 | |
| 投资金额 + 时间 | |
| 与企业的关系 | 早期天使？战略资源引入？家族朋友？ |
| 其他知名投资项目 | 还投过哪些公司？ |
| 公开表达的观点 | 对行业/企业的看法 |

### 5.3 投资逻辑分析框架

针对每个关键投资人回答以下问题：

```
Q1: 该投资人的投资阶段偏好是什么？
    （早期成长期/晚期Pre-IPO/全阶段？这次投资是否符合其常规偏好？）

Q2: 该投资人在这个赛道的布局情况？
    （是首次入场还是持续加注？还投了哪些同类企业？是否存在利益冲突？）

Q3: 投后参与程度如何？
    （有董事会席位吗？参与了哪些关键决策？派驻了人员吗？）

Q4: 投资人对企业的公开评价？
    （媒体报道中投资人的表态？看好什么？担心什么？）

Q5: 投资人自身的处境和压力？
    （基金存续期临近？需要退出案例？LP压力？这会影响其对被投企业的态度）
```

## 6. 人物关系网络识别

### 6.1 关系类型与发现方法

| 关系类型 | 发现方法 | 价值 | Query示例 |
|---------|---------|------|----------|
| **校友关系** | 教育背景交叉比对 | 高信任度合作基础 | `"{人名A}" "{人名B}" 校友 同学 {校名}` |
| **前同事关系** | 履历交叉比对 | 了解内部运作方式 | `"{人名A}" "{人名B}" 同事 共事 {前公司}` |
| **同乡关系** | 籍贯信息交叉 | 地缘信任网络 | `"{人名A}" 籍贯 家乡 出生地` |
| **协会会员** | 行业协会名单 | 正式协作关系 | `"{行业协会}" 会员单位 理事 {公司名}` |
| **同期创业** | 创业时间线比对 | 创业圈社交重叠 | `"{年份}" 创业浪潮 {行业} 创业者` |
| **投资人与被投** | 融资信息 | 资本纽带 | 见第5节 |
| **兼任关联** | 工商/公告 | 显性关联 | `"{人名}" 兼职 董事 监事` |

### 6.2 关系网络图构建原则

```
节点 = 人物（person）或组织（company/investor）
边 = 关系类型（带方向和强度）

强度分级：
  ★★★★★ 股权控制 / 亲密合伙创业
  ★★★★☆  董事会层面长期合作
  ★★★☆☆  校友 / 前同事 / 投资关系
  ★★☆☆☆  同一协会 / 同期创业 / 同乡
  ★☆☆☆☆  公开场合同台（弱信号，需谨慎解读）
```

### 6.3 关系网络的报告呈现

在报告中不以图形方式呈现（PDF限制），而是用文字描述关键关系链：

```
格式："A —[关系]→ B —[关系]→ C"

示例："吴彬(MC机构背景) —[KK集团创始人]→ KK集团 —[京东/经纬投资]→ 张磊(京东系)"
      "叶国富(湖北籍) —[名创优品创始人]→ 名创优品 —[腾讯/高瓴投资]→ ..."
```

重点关注以下模式并标注：
- **三角关系**：三人/三公司之间形成闭环关联（可能暗示协同或利益输送）
- **桥接人物**：同时连接多个独立圈子的人（信息枢纽和资源调度中心）
- **隔离信号**：目标企业核心人物与行业主流圈子无明显关联（可能是劣势也可能是独立性标志）
