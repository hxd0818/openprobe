## 1. 核心思维

### 旧版问题

| 问题 | 表现 |
|------|------|
| 固化query列表 | 4节内容全是query模板+提取表格，Agent机械执行 |
| 无结构诊断 | 不区分"股权高度集中的家族企业"和"分散持股的上市公司"，用同一套query |
| 缺推理链 | 直接给搜索词，没有让Agent先想"这家公司的资本关系可能长什么样" |

### 新版框架

```
Step 1: 资本结构诊断
    "这家公司的钱从哪来？谁在控制？结构复杂还是简单？"
    
Step 2: 关键节点识别
    "哪些资本关系的节点是关键的？哪些信息是公开可获取的？"
    
Step 3: 自主规划
    Agent 针对每个节点自己决定怎么查
    
Step 4: 执行 + 动态调整
    
Step 5: 图谱输出 + 风险评估
```

## 2. Step 1：资本结构诊断

> 基于Phase 0数据，Agent 对目标企业的资本结构做出初步判断。

### 资本结构五问

| # | 问题 | 回答后得到的信息 |
|---|------|-----------------|
| Q1 | **谁控制这家公司？** | 控股股东/实控人/最终受益人方向 |
| Q2 | **控制链条多长？** | 简单（自然人直接控股）vs 复杂（BVI→HK→境内多层嵌套/VIE） |
| Q3 | **钱从哪来？** | 创始人自有/VCPE/国资/上市公司/混合？ |
| Q4 | **对外投资活跃吗？** | 有大量子公司/被投企业？还是纯运营体？ |
| Q5 | **资本关系复杂度预判** | 低（简单控股）/ 中（有投资版图）/ 高（复杂集团/VIE/一致行动人） |

### 输出格式

```markdown
## [目标企业名] 资本结构诊断

### 控制权概貌
[谁在控制？通过什么路径？]

### 融资历史概貌
[融过几轮？谁投的？到什么阶段了？]

### 投资版图概貌
[对外投资多吗？主要投向什么领域？]

### 结构复杂度: [低/中/高]
[判断理由]

### 关键调查方向（Agent自填）
1. [最需要查清楚的资本关系]
2. [其次重要的]
3. [如果有余力再查的]
```

## 3. Step 2：识别关键节点与信息可及性

### 节点类型

| 节点类型 | 定义 | 示例 |
|---------|------|------|
| 🔴 **控制权核心** | 实控人/控股股东/控制链条上的关键实体 | BVI壳公司、境内WFOE、VIE协议方 |
| 🔴 **重大资金来源** | 大额投资方/债权人/重要融资渠道 | 主要VC/PE、银行授信、发债 |
| 🟡 **重要投资标的** | 子公司/联营/被投企业中的主要实体 | 核心业务子公司、跨界投资 |
| 🟡 **关联方网络** | 高管兼任/同地址/家族/一致行动人 | 兼职董事、家族成员持股公司 |
| 🟢 **一般参股/小股东** | 占比低、影响小的投资关系 | 小额财务投资者 |

### 信息可及性预判

| 可及性 | 特征 | 典型来源 |
|--------|------|---------|
| **高** | 上市/拟上市/国企 | 招股书、年报、公告、工商公示系统 |
| **中** | 有融资历史的民企 | 融资报道、工商变更、企查查/天眼查摘要 |
| **低** | 非上市且无近期融资 | 间接线索（诉讼、招聘、新闻提及） |
| **极低** | 复杂VIE/BVI嵌套 + 长期无披露 | 只能确认结构存在，细节可能无法获取 |

## 4. Step 3：自主规划发现策略

### 规划要求

```markdown
## 资本关系调查计划

### 资本结构判断摘要
[2-3句话：控制权在哪、复杂度如何、关键风险点]

### 本轮次规划
**Round X（资本专项）：**
- 目标节点：[具体要查什么]
- 选择的方法：[Agent自定]
- 选择理由：[为什么有效]
- 预期产出：[希望获得什么]
```

### 自主规划的指导原则（非强制）

**原则1：从确定性最高的信息入手**
- 上市/拟上市 → 招股书的"历史及重组""控股股东"章节 = 最完整准确的资本关系描述
- 有工商变更记录 → 变更记录揭示股权变动时间线
- 有融资报道 → 投资方名单 + 金额 + 估值

**原则2：顺藤摸瓜（从已知到未知）**

```
已知实控人姓名 → 追踪其名下其他公司 → 发现隐性关联
已知投资方名称 → 追踪其投资组合 → 发现同赛道被投企业（=潜在竞对）
已知子公司名称 → 追踪其股东 → 可能发现合资方/隐名股东
```

**原则3：关注异常信号**

| 异常信号 | 可能隐藏的信息 | 追查方法 |
|---------|---------------|---------|
| 频繁法人变更 | 控制权不稳定或代持 | 搜"公司名 法人 变更 历史" |
| 短期大额增资 | 新投资人进入或资产注入 | 搜"公司名 增资 注册资本 变更" |
| 高管密集兼职 | 隐性的商业网络 | 逐个追踪高管的其他任职 |
| 同地址注册多家公司 | 关联方或代持平台 | 搜地址 + "注册公司" |
| 诉讼中的未知方 | 隐藏的商业关系 | 从判决书当事人反查 |

**原则4：资本关系与其他维度的联动**

```
资本 → 竞对:  同一投资方投了谁？（可能=竞对）
资本 → 产业链:  投资方是否也是供应商/客户？
资本 → 人物:  投资人/董事的人际网络
人物 → 所有维度:  高管兼职是跨维度关联的核心入口
```

## 5. Step 4：执行与动态调整

### 每轮记录

```markdown
### Round X 执行记录（资本维度）
- **目标**: [本轮查什么]
- **方法**: [实际用了什么]
- **发现**: [新实体/新关系/新事件]
- **图谱更新**: [capital_graph 是否需更新]
- **下一步**: [还缺什么]
```

### 收敛判断

| 条件 | 说明 |
|------|------|
| 🔴 控制权链条已完整（从自然人到目标公司） | ✅ 必须 |
| 🟡 主要投资方(>5%股权)已识别 | ✅ 必须 |
| 🟡 重要子公司/被投企业已列出 | 尽力 |
| 🟡 高管兼任网络已初步绘制 | 尽力 |
| 🟢 一般参股已登记 | 有则更好 |

| C02 | 向下投资追踪 | 有子公司的企业 | `"公司名" 全资子公司 控股子公司 分公司 对外投资` |
| C03 | 融资历史查询 | 有融资历史的企业 | `"公司名" 融资 A轮 B轮 C轮 D轮 IPO 估值 投资方` |

| C05 | 高管/董事网络 | 所有企业 | `"公司名" 董事会 监事 高管团队 核心人员` |
| C06 | 单人追踪 | 已知关键人物姓名 | `"{姓名}" 兼职 任职 董事 监事 投资 公司` |
| C07 | 同地址/园区 | 制造业/集群行业 | `"{地址}" 公司 注册 企业 邻近` |

| C09 | 协议控制/VIE | 互联网/海外架构 | `"公司名" VIE 一致行动 协议控制 委托投票` |
| C10 | 同投资人线索 | 已知投资方时 | `"{投资方}" 投资组合 被投企业 portfolio` |

| C14 | 债务/担保查询 | 关注财务健康时 | `"公司名" 债务 担保 逾期 诉讼 债权人` |

## 8. 与其他维度的交叉关联

| 方向 | 方法 |
|------|------|
| 资本 → 竞对 | 同一投资方的被投企业 = 间接竞对候选 |
| 资本 → 供应链 | 投资方可能是客户或供应商；子公司可能在产业链上 |
| 资本 → 人物 | 高管/投资人是资本网络的节点，也是人物维度的核心对象 |
